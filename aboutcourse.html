Código del curso: IC8057

Nombre del curso: Introducción al Desarrollo de Páginas Web

Período del curso: Verano 2019

Tipo de curso: Electivo

Créditos: 3

Horas por clase: 11 por semana

Horas extraclase: 22 por semana

Asistencia: Obligatoria

Requisitos:
- IC4301 Bases de Datos I
- IC4700 Lenguajes de Programación (Preferible)

Profesora del curso: Ericka Solano Fernandez

Objetivo General del curso: Conocer los conceptos básicos del desarrollo de aplicaciones para Internet y desarrollar aplicaciones sencillas basadas en páginas y/o componentes.

- Concepto Historico
- Conceptos y Terminología básicos
- Arquitectura y componentes
- Patrones de diseño para facilitar el desarrollo WEB
- Herramientas y tecnologías para el desarrollo WEB
- Frameworks de trabajo WEB
- Integración con bases de datos
- Metodologías ara desarrollo de aplicaciones WEB
